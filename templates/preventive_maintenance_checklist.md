# Preventive Maintenance Checklist

## Daily Checks
- [ ] Review power alerts
- [ ] Review cooling alerts
- [ ] Check temperature and humidity exceptions
- [ ] Review unresolved critical tickets
- [ ] Confirm access/security exceptions

## Weekly Checks
- [ ] Review UPS/generator status
- [ ] Review rack utilization
- [ ] Review power load balance
- [ ] Review cooling capacity
- [ ] Review recurring incidents
- [ ] Update maintenance schedule

## Monthly Checks
- [ ] Test escalation process
- [ ] Review SLA performance
- [ ] Review audit logs
- [ ] Review capacity forecast
- [ ] Review cost and utilization trends
- [ ] Update improvement backlog
